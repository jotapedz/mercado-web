# Mercado Web (React + Material UI)

Frontend web para gerenciamento de mercado com:

- cadastro de clientes
- cadastro de produtos
- reposicao de estoque
- registro de vendas
- dashboard com indicadores
- integracao com API ASP.NET Core

## Requisitos

- Node.js 20+ (ou 18+)
- npm
- API backend rodando em `http://localhost:5043`

## Como rodar

```bash
npm install
npm run dev
```

Abra o endereco mostrado no terminal (normalmente `http://localhost:5173`).

## Observacao

Os dados agora sao persistidos no PostgreSQL via backend.
